# JCI

Javascript Continuous Integration

In the beginning,Only used in fanhua

功能规划:

* markdown 工程的持续集成
* node 工程的持续集成
  1. 自动化输出接口文档
  2. bug收集提醒
  3. 集成/单元测试
* web 工程的持续集成

## 定义

### JCI 实例

一系列 Tast 任务的集合

### Task 任务

事件发生之前定义的一系列需要执行的 Work 有序数组

### Work 工作
